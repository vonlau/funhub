Advanced Search
Find users based on multiple criteria at once (e.g. "active users in Malaysia with 100+ points"), then export or run bulk actions on the results.
⚙️ How it works
  • Set one or more filters and click Search.
  • All filters work together — users must match every condition set.
  • Your search is remembered so export and bulk actions use the same result set.
  • The results table is empty until a search is run (to avoid loading all users by accident).

Bulk actions after search

💡
Background jobs: Large bulk actions are processed in the background. You get a job ID and can check its status (processing / completed / how many done).
Support
🎫
Support Tickets
Tickets submitted by users (e.g. "I didn't receive my reward"). Each has a status (open / in progress / resolved), an assignee, and a category. Reply to the user and mark resolved when done.
❓
FAQs & FAQ Categories
Create and manage frequently asked questions shown in the app's help section. Organise them into categories (e.g. "Account", "Payments") to reduce support tickets.
🗂️
Support Request Categories
Define categories for tickets (e.g. "Technical", "Billing") to route and report on them.
