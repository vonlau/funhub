Points Settings
Global rules — how points are earned, expiry policy, and earning caps. These apply consistently to all users.
Reward Building Blocks
🧩
Reward Components
The building blocks used in Campaigns. Each component defines what a user receives — points, a voucher, a badge, or a promo code. You create them once here, then reuse them across campaign reward pools, collectible requirements, and mission rewards.
Used inHow
Campaign Reward PoolEach prize in a lucky draw is a reward component (e.g. "Win 100 points")
Collectible Set completionThe reward users get when they finish collecting a full set
Mission task rewardsPoints or draw chances given when a task is completed
💡
Think of them as prize templates. Create a reward component once here, then pick it anywhere a reward is needed.
Promotion Codes
🎫
Promotion Codes
Codes users enter in the app to receive a benefit (e.g. 100 points). Set what the code gives, its expiry, and usage limit. The system enforces limits automatically.
📁
Promotion Code Groups
Organise codes into groups by campaign. Useful for bulk exports and tracking usage. Code groups can also be used as prizes in campaign reward pools.
