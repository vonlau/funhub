Notification Templates
📝
What Are Templates?
A reusable message you write once and attach to a specific action (e.g. "after buying a FunCard"). The system fills in placeholders like {{user_name}} with real data for each user.
FieldWhat it means
NameYour internal label (e.g. "Fun Card Purchase Confirmation")
CodeUnique short name used by the system (e.g. product_purchase)
CategoryOptional grouping (e.g. "purchase", "reward")
Title (EN/CN)Notification heading — can include placeholders
Message (EN/CN)Body text — can include placeholders
URLOptional deep link (e.g. to order history)
ActiveOnly active templates appear when setting up FunCards or packages
ℹ️
Templates are only used in two places: FunCard (product) purchases and Coin Package purchases. All other notifications use their own separate system.
🎴
FunCard Purchase Template
Attached to a FunCard under Sales → Products → Enable Purchase Notification.
Available placeholders
PlaceholderWhat it inserts
{product_name}Name of the FunCard bought
{user_name}The user's name
{amount}Amount paid
{transaction_no}Transaction reference number
ℹ️
Example: "Hi {user_name}, your purchase of {product_name} for RM {amount} is confirmed. Ref: {transaction_no}."
💡
No template? A default "Fun Card Purchase Successful" message is sent automatically.
📦
Coin Package Purchase Template
Attached to a Coin Package under Sales → Coin Packages → Enable Purchase Notification. A template is required when this is turned on.
Available placeholders
PlaceholderWhat it inserts
{package_name}Name of the coin package
{user_name}The user's name
{amount}Amount paid
{base_coins}Base coins in the package
{bonus_coins}Bonus coins received
{total_coins}Total coins credited
{transaction_no}Transaction reference
ℹ️
Example: "You received {total_coins} coins ({base_coins} + {bonus_coins} bonus). Thank you, {user_name}!"
