App Configuration
Core settings: app name, default language, currency, and feature flags that control what is enabled across the whole platform.
🛠️
Maintenance Mode
Put the app into maintenance mode to prevent user access during updates or fixes. Shows a maintenance message to users.
🔗
Connected Applications
Third-party integrations and API keys linked to the platform (e.g. payment gateways, SMS providers).
