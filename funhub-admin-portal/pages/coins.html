Coins (Fun Coins)

Fun Coins are the in-app currency — earned from purchases, spent on rewards. Manage balances and global coin rules here.

Coin Management
📋
Coin Balances
See every user's total coin balance, amount purchased, amount spent, and last activity. Open any user to view their full coin transaction history.
➕➖
Adjust Coin Balance
Manually add or deduct Fun Coins from a specific user — for example to correct an error or resolve a support case.
1
Find the user

Go to Coin Management and search for the user.

2
Click Adjust

Choose Add or Deduct.

3
Enter amount

Type the number of coins.

4
Add a reason

Optional note for records (e.g. "Support correction — ticket #123").

5
Confirm

Applied immediately and recorded in the user's coin history.

⚠️
Every adjustment is logged. The system records who made the change and when. Only use this for genuine corrections — not as a shortcut for rewards or campaigns.
